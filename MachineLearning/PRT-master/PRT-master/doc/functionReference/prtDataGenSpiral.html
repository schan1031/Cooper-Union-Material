prtDataGenSpiral
 prtDataGenSpiral   Generates some spirally clustered example data 
 
   DATASET = prtDataGenSpiral returns a prtDataSetClass with randomly
   generated data in a spiral pattern in 2 dimensions.
 
   Example:
 
   ds = prtDataGenSpiral;
   plot(ds)
