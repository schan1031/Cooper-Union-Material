prtDataGenCircles
 prtDataGenCircles   Generates circularly clustered data 
 
   DATASET = prtDataGenCircles returns a prtDataSetClass with randomly
   generated data in a circular pattern.
   
   Example:
 
    dataSet = prtDataGenCircles;
    plot(dataSet);
